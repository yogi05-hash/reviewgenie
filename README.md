# 🧞 ReviewGenie — AI Review Responder

AI-powered review reply tool for small businesses. Never miss a review again.

## 🚀 Deploy in 60 seconds

### Option 1: Vercel (Recommended)
1. 📂 Push this repo to GitHub
2. 🔗 Go to [vercel.com/new](https://vercel.com/new) → Import your repo
3. 🔑 Add environment variables from `.env.example`
4. 🚀 Click **Deploy**

### Option 2: Terminal
```bash
npm install
vercel --prod
```

## 💻 Local Development
```bash
npm install
npm run dev
# Open http://localhost:3000
```

## 📦 Tech Stack
- Next.js 15 + TypeScript
- Tailwind CSS v4
- Vercel hosting
